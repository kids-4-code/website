# Kids for Code website

The new Kids for Code website, written using Vue.js 3 & Tailwind CSS.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
