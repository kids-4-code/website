About Kids for Code

Kids for Code teachers are high school students who are passionate about computer science. We are experienced and qualified in the computer science field, with many of our teachers winning awards and creating apps.

In today‚Äôs increasingly technology-centered world, computer science is an ever-expanding and practically essential skill. We want to teach and prepare kids to become active and engaged citizens in the future, by providing them access to computer science skills not taught in schools.

We truly believe that by introducing computer science to children at an early age, we are setting them up for a brighter future-a future that they can code themselves-and creating a generation that is innovative, passionate, and driven. After all, it‚Äôs never too early to start learning!

Classes for the 2023 Summer session will run for 8 weeks, from Sat, July 1st to Sun, August 20th.

Python

Python 1A

Sundays, 12:30-1:15pm EST | 6+ y/o

A beginner level course aimed at children ages 6 & up. It provides an gentle introduction to Python & text-based coding.

Python is a general purpose scripting language that's popular in data science, AI, and backend programming.

In Python 1A, students will learn basic Python syntax & programming concepts, like variables, conditional statements, data types, etc.

Learn more

Python 1AB

Sundays, 2-2:45pm EST | 8+ y/o

Python 1AB is a more advanced, faster-paced version of Python 1A that covers all the content in 1A as well as a few additional topics.

Python is a general purpose scripting language that's popular in data science, AI, and backend programming. In Python 1AB, students will learn basic Python syntax & programming concepts, like variables, conditional statements, data types, etc.

Some of the additional topics covered in Python 1AB include scope, while loops, for loops, etc.

Learn more

Python 2

Sundays, 3-3:45pm EST | 8+ y/o

Python 2 builds on the material covered in Python 1A/1AB, taking students on a deeper journey into the Python programming language. It includes crucial concepts like functions, lists, string manipulation, and file I/O that are essential for more complex applications.

Python 2 will give students a solid foundation in these universal programming concepts which can then be applied to many other popular programming languages.

Learn more

Web Design

Web Design 1A

Saturdays, 1-1:45pm EST | 6+ y/o

Introduces students to the basic principles of web development. Students will learn how to develop and style basic websites using HTML & CSS.

Topics covered in 1A include HTML page structure, basic CSS styling, and the CSS box model. Students will able to create their own static (no JavaScript) website by the end of the class.

Learn more

Web Design 1AB

Saturdays, 2:30-3:15pm EST | 8+ y/o

Python 1AB is a more advanced, faster-paced version of Python 1A that covers all the content in 1A as well as a few additional topics.

Introduces students to the basic principles of web development. Students will learn to develop and style basic websites using HTML & CSS.

In addition to the topics covered in 1A: HTML page structure, basic CSS styling, and the CSS box model, 1AB will learn about more advanced CSS styling and a few additional HTML tags. Students will able to create their own static (no JavaScript) website by the end of the class.
